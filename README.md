# Bio-Inspired Robotic Fish Simulator

A physics-based simulator for analyzing propulsion and efficiency of a bio-inspired robotic fish.

This project models fish swimming using simplified hydrodynamics and tail-beat propulsion.

## Features

- Tail-driven propulsion model
- Hydrodynamic drag simulation
- Swimming speed vs tail frequency analysis
- Cost of transport evaluation
- Strouhal number analysis
- 2D animation of fish locomotion
- 3D visualization of rigid-body fish model
